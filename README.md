# Handmade Jewelry – Responsive Landing Page

This is a collaborative team project developed during the **GoIT** Fullstack Developer course. Our team of developers implemented this modern, high-performance landing page based on a provided Figma design, focusing on clean code, semantic HTML, and flawless responsiveness.

## 🔗 Live Page
Check out the live demo here: [View Project](https://vitalik0422.github.io/NGD-team-project/)

## 🛠 Tech Stack
* **HTML5 / CSS3** – Semantic markup and modern styling.
* **BEM Methodology** – For structured and readable class naming.
* **Vite** – As a fast and modern build tool.

## 🌟 Key Features
* **Pixel Perfect Approach:** Precise implementation of the provided design across all sections.
* **Fully Responsive:** Optimized for Mobile (375px+), Tablet, and Desktop versions.
* **Mobile-First Workflow:** Developed starting from the smallest screens for better performance.
* **Interactive Elements:** Functional mobile navigation menu and validated contact form.
* **Optimized Assets:** Images are optimized for fast loading without losing quality.

## 👥 Team Collaboration
Our team worked together using the **Scrum** methodology:
* **Team Lead:** Managed tasks, conducted code reviews, and ensured project consistency.
* **Scrum Master:** Facilitated daily stand-ups and kept the workflow on track.
* **Developers:** Responsible for building individual UI components and sections.
